CURSO GIT E GITHUB - ALURA

git init
	- inicializar repositório
git status
	- mostra informações sobre o repositório
git config --local user.name "Seu nome aqui"
	- configurar username
git config --local user.email "seu@email.aqui"
	- configurar o email
git add .
	- adicionar todos os arquivos para serem monitorados pelo git
git commit -m "mensagem"
	- comitar os arquivos, com a mensagem enre aspas
